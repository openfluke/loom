# Windows ARM64 native builds

Use **one script** from WSL/Linux:

```bash
cd loom/welvet/cabi/internal/build
./build_windows_arm64.sh --skip-lib
```

Full guide: [`README_WINDOWS_ARM64.md`](../welvet/cabi/internal/build/README_WINDOWS_ARM64.md)

Do **not** use `./build_unix.sh windows arm64` — it redirects here.
